# Raspberry-hub
Pro správné fungování všech aplikací, je nutné být registrován v síti Tailscale.

Build mobilní aplikace i aplikace na chytré hodinky jsou ve složce Builds.

Pro přístup k funkční webové aplikaci je nutné být registrován v systému firebase.

Desktopovou aplikaci spravující Raspberry Pi lze spustit přes soubor App.py, ale aplikace bude fungovat jen na zařízeních s Linux distribucí.
Před spuštěním bude nutné nainstalovat několik knihoven.
